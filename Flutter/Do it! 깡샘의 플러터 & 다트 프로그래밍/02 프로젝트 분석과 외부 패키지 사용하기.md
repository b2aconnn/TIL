# 플러터 프로젝트 분석하기

## 프로젝트 폴더 구성

- android : 안드로이드 앱 구성
- ios : ios 앱 구성
- lib : 다트 파일
- test : 테스트 다트 파일

<br>

### 프로젝트 파일 구성

- lib/main.dart : 앱의 메인 다트 파일
- .gitignore : 깃에 업로드하지 않을 파일 등록
- pubspec.yaml : 플러터 프로젝트의 메인 환경 파일

<br>

### main.dart 파일 분석하기

- import 구문
  - 다른 다트 파일을 불러올 때 사용
- main() 함수
  - 다트 엔진의 진입점이며, 다트 엔진이 main() 함수를 호출하면서 앱이 실행됨

<br>
